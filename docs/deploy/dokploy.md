
# 使用 Dokploy 部署

Dokploy 是一个开源的 PaaS 平台，可以让你在自己的服务器上轻松部署应用。

## 前提条件

你需要根据 Dokploy 的官方文档，在你的服务器上安装 Dokploy 服务。

## 部署步骤

1.  **构建 Docker 镜像**:
    确保你已经根据前面的文档，成功使用 Github Actions 远程构建了 Docker 镜像。

2.  **在 Dokploy 中创建项目**:
    在 Dokploy 的部署面板中创建一个新项目。

3.  **配置项目**:
    -   **自定义域名**: 输入你想要绑定的域名，可以是根域名或子域名。
    -   **部署路径**: 使用根目录作为部署路径。
    -   **容器端口**: 使用默认的 `3000` 作为容器端口。
    -   **HTTPS**: 打开 HTTPS 选项，Dokploy 会自动为你的域名生成 SSL 证书。

4.  **拉取镜像并部署**:
    配置完成后，从你的 Docker 仓库中拉取镜像并进行部署。
